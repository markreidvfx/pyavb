pyavb
=====

A python module for the reading and writing Avid Bin Files (AVB) files.

Requirements
------------

- Python >= 2.7

Installation
------------

clone the latest development git master::

    git clone https://github.com/markreidvfx/pyavb
    cd pyavb
    python setup.py install
